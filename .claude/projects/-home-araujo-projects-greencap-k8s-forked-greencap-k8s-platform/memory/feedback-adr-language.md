Escrever novos ADRs em português.

**Why:** O CLAUDE.md especifica inglês apenas para comentários no código. ADRs são documentação — o usuário prefere português para facilitar leitura pelo time.

**How to apply:** A partir do ADR-0002, todos os novos ADRs criados em `docs/adr/` devem estar em português.
